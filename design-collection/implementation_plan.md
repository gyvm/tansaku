# 実装計画

## 変更の目的
- Design Collection 内に「録音ボタンのデザイン案」を1画面で大量に比較できるページを追加する。
- 各ボタンで「録音開始前（押すと開始）」と「録音中（押すと停止）」を実際にトグルできる状態を実装する。
- モダン/シンプル/カラフル/アニメーション付きなど、幅広いバリエーションを20〜30個用意する。

## ユーザーレビューが必要な項目
- 25案の中で優先的に深掘りしたいデザイン（色・形・ラベル表現）。
- ラベル文言の統一方針（日本語中心か英語併記か）。
- アニメーション強度（より控えめ/よりリッチ）。

## 変更内容の詳細
1. `src/pages/recording-buttons/App.tsx` を新規作成。
   - 25種類の録音ボタンスタイル定義を配列で管理。
   - 各カードで独立した録音状態トグルを持たせ、クリックで開始/停止を切り替え。
   - 状態に応じてアイコン・色・文言・アニメーションを変更。
2. `src/main.tsx` に新規ルート `/recording-buttons` を追加。
3. `src/pages/Home.tsx` に新規カードを追加し、一覧から遷移可能にする。

## 検証計画
- `npm run build` を実行し、型チェックとビルド通過を確認する。
- 開発サーバーを起動し、`/recording-buttons` で以下を確認する。
  - 25個のカードが表示される。
  - 各カードが個別に開始/停止を切り替えられる。
  - 録音中の視覚差分（ラベル・色・アイコン・アニメーション）が確認できる。
- 画面スクリーンショットを取得する。
