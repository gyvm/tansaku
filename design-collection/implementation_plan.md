# 実装計画

## 変更の目的
録音開始・停止・録音中ステータスが複数箇所に分散していたため、統合コントロールとして1箇所に集約し、視線移動と操作の一貫性を高める。加えて、設定ボタンや録音開始/停止の操作に連動してUIが動くようにし、操作の手応えを明確にする。

## ユーザーレビューが必要な項目
- 統合コントロールの配置（セッションカード上部・タイトル右側）で操作がしやすいか
- 状態バッジと停止ボタンの組み合わせが意図に合うか

## 変更内容の詳細
- セッションカードを「タイトル/編集」「2チャンネル波形」「操作ボタン」「タイマー」の縦構成に再編
- マイク/システム音声の2段波形と LIVE/MONITORING 表示を追加
- 中央の停止ボタン＋左右の一時停止/マーカー操作に集約
- 録音開始/停止/一時停止でステータス、タイマー、波形アニメーションが連動するように実装
- 設定ボタンを押下した際に動きが出るようにホバー/アクティブのモーションを追加
- ボタンサイズや影、背景色を調整してポップな質感に寄せる

## 検証計画
- `npm run dev` で `http://localhost:5173/meeting-recorder-macos` を表示
- 録音コントロールが1箇所に集約されていることを確認
- レイアウト崩れがないことを確認
- 録音開始/停止でUIの状態が切り替わることを確認
