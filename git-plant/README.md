A CLI tool that visualizes your Git branch structure as a growing plant in the terminal.

## Features
- **Monochrome & Emoji**: Clean ASCII art structure with emoji leaves.
- **Abstract Visualization**: Focuses on branch lifespans and relationships, not every commit.
- **Interactive**: (Coming Soon) TUI to navigate branches.

## Installation

```bash
pip install .
```

## Usage

```bash
git-plant
git-plant --help
```
